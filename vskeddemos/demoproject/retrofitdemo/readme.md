
# retrofit技术示例

接口请求发送技术示例  

目录结构  
├─.idea  
└─src  
    ├─main  
    │  ├─java  
    │  │  └─com  
    │  │      └─jat  
    │  │          └─http  请求发送接口与实现类  
    │  └─resources  日志配置文件  
    └─test  
        └─java  
            └─com  
                └─jat  
                    ├─http 单元测试  
                    └─test 单元测试示例  


本技术示例用来演示使用retrofit技术发送get,post请求  
具有动态地址构建，单元测试，日志打印等技术集成。
