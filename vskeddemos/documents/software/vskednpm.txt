
在使用npm以前请先安装nodejs

1* 查看版本号
npm -v

2* 当前可以使用的命令
npm help

3*查看命令帮助
npm help [命令]
npm help install

4* 升级当前npm
npm install npm -g

5* 搜索模块
npm search [模块名]
npm search 

6* 初始化package.json
npm init

7* 安装依赖包
npm install [模块名]
npm install jquery
npm install webpack -g
添加参数-g为全局

8* 更新模块
npm update [模块名]
npm update express

9* 更新缓存
npm cache clear

10* 启动应用
npm start












pm2常用命令

安装pm2
npm install -g pm2

以下是PM2的常用命令

```
pm2 list # 查看当前正在运行的进程
pm2 start all  # 启动所有应用
pm2 restart all  # 重启所有应用
pm2 stop all # 停止所有的应用程序
pm2 delete all # 关闭并删除所有应用
pm2 logs # 控制台显示所有日志

pm2 start 0  # 启动 id为 0的指定应用程序
pm2 restart 0  # 重启 id为 0的指定应用程序
pm2 stop 0 # 停止 id为 0的指定应用程序
pm2 delete 0 # 删除 id为 0的指定应用程序

pm2 logs 0 # 控制台显示编号为0的日志
pm2 show 0  # 查看执行编号为0的进程
pm2 monit mall4j-pc # 监控名称为mall4j-pc的进程
```
