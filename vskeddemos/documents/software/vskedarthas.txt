
第一步下载arthas-boot.jar文件。然后运行

java -jar arthas-boot.jar

运行完成后选择一个java进程

执行追踪，也可以是controller层
trace com.yourproject.xxx.xxxController list

详细追踪
trace org.ruoyi.system.service.impl.xxxServiceImpl list -n 1

trace com.baomidou.mybatisplus.core.mapper.BaseMapper selectVoPage -m 1000

查看执行的sql
watch org.ruoyi.system.mapper.SysApiLogMapper selectVoPage '{params,sql}' -x 2

# 监控 selectPageApiLogList 方法的调用次数和耗时
monitor org.ruoyi.system.service.impl.SysxxxServiceImpl selectPageApiLogList -c 5

# 查看方法调用堆栈
stack org.ruoyi.system.service.impl.SysxxxServiceImpl selectPageApiLogList
