开发指南

本开发指南将介绍如何使用灵云SDK Java API，您可以按照如下步骤来熟悉灵云SDK的使用方法：

  • 环境准备
    列出了在使用灵云SDK之前应该做的准备工作。
  • 简单示例
    以HWR识别为例，展现了HWR SDK一次完整的功能调用过程。
  • 各能力通用流程
    介绍了各SDK使用时的共同点，比如SDK初始化/反初始化, 新建会话/停止会话等。

灵云SDK包含四类API:

  • 系统API ：提供统一配置管理、授权管理、本地资源管理等不同能力所需的通用操作
  • 能力API ：提供各种不同的智能人机交互(HCI)技术的操作，如识别、合成等
  • 扩展API ：提供构建在HCI能力API之上的，更为简单方便的调用接口，一般和硬件设备结合，隐藏具体设备操作

目前灵云SDK提供的HCI能力如下：

  • TTS能力 (语音合成)
  • ASR能力 (语音识别)
  • HWR能力 (手写识别)
  • OCR能力 (光学字符识别)
  • MT能力 (机器翻译)
  • NLU能力 (意图识别)
  • VPR能力 (声纹识别)
  • KB能力 (键盘识别)

目前灵云SDK提供的扩展API如下：

  • TTS播放器
  • ASR录音机
  • OCR拍照器
